GitHub is meaningful to me

You hold product and process

Aside from being the world's biggest collection of open-source software, version-control allows users to follow its development. You're an invaluable resource for learning and building.

Social coding

Helping people work together. GitHub is code, but ultimately you're removing friction from and innovating collaboration. From NASA to Sub-Saharan Africa, you're making the web more open

Democratizing open source development

I'm not sure Apple would've made Swift open-source if it wasn't for GitHub. And Java certainly wouldn't be as ubiquitous. GitHub helped make open-source the default.
