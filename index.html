GitHub is meaningful to me

You hold product and process

Aside from being able to browse the world's biggest collection of open-source software, users can also follow its development - like a library with version-control. You're an invaluable resource for learning and building.

Social coding

I see GitHub's primary goal as helping people work together. GitHub is code, but ultimately you're innovating and removing friction from collaboration. From NASA to Sub-Saharan Africa, you're making the web more open and team workflows more efficient.

Democratizing open source development

I'm not sure Apple would've made Swift open-source if it wasn't for GitHub. And Java certainly wouldn't be as ubiquitous. GitHub helps make open-source the default.

I know your product

I've used GitHub to manage bug reports, to manage engineers, to manage my own code, to code collaboratively. I've used it to find and learn new libraries, to pick the d3 layout to use, to see what my friends are building.

I know what to take seriously

I feel as comfortable positioning products as I do writing AJAX post requests as I do designing mockups. I'm a learn-it-all - I follow my curiosities. Being too comfortable is uncomfortable to me.

I take my work seriously. I take my team seriously. I think honesty, respect and accountability are non-negotiables.
But I don't take myself too seriously.
